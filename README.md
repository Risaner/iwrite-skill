# Score-Forge ⚒️

**机器批改英语作文高分锻造策略**

针对 iWrite（外研社）和批改网（pigai.org）两大主流英语作文机器批改系统的高分写作策略 Skill。

## 这是什么？

基于对机器评分系统**语料库距离算法**的逆向理解，总结出的一套可复现的高分写作策略。经过实战验证（CET-6 议论文 + 图表描述文），稳定达到 **4×5星 + 1×4星** 的成绩。

## 核心原理

机器批改系统不真正"读懂"内容，它通过 **192+ 个语言分析维度**，将你的作文与母语者标准语料库对比，计算"距离"。距离越近，分越高。

## 六大策略

| 策略 | 提分效率 | 说明 |
|------|---------|------|
| 字数超量 | ⭐⭐⭐⭐⭐ | 超要求 30%-50%，但绝不超过上限 |
| 结构词轰炸 | ⭐⭐⭐⭐⭐ | Firstly/Moreover/In conclusion 铺满每段 |
| 关键词复现 | ⭐⭐⭐⭐ | 题目核心词全文出现 5-8 次 |
| 高级词汇替换 | ⭐⭐⭐⭐ | think→maintain, important→crucial |
| 加分句式 | ⭐⭐⭐ | 倒装/虚拟/强调/非谓语 |
| 格式零失误 | ⭐⭐⭐ | 标点后空格，零拼写错误 |

## 实战案例

### CET-6 AI 议论文
- 要求 180 词 → 实写 270 词
- 结果：语言5星 / 内容5星 / 篇章结构4星 / 技术规范5星
- 反馈：能使用高级词汇、全文流畅连贯、句式丰富复杂

### F City Poverty Alleviation 图表描述文
- 要求 180 词 → 实写 250 词
- 结果：同样 4×5星 + 1×4星
- 教训：尝试加总领句优化篇章结构 → 超字数扣分 → 过度优化反而降分

## 血泪教训

1. **超字数上限 = 硬扣分**，比任何技巧收益都大
2. **过度堆砌高级词 → "句子不通顺"扣分**，自然 > 华丽
3. **4×5星 + 1×4星 是实战天花板**，追求稳定而非完美
